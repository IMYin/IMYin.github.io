通过sshfs来将远程目录挂载到本地¶

sshfs，就是可以把ssh连接到的主机资源，映射到本机的文件系统当中，然后用户可以像操作本地文件一样操作，而实际的文件改动将通过ssh传输到远程主机当中。类似于mount了一个windows共享目录，或者另外一台linux主机的samba共享，或者是nfs共享，只是“共享协议”变成了ssh通道罢了

shell if 语句在cron中不生效

背景：想要执行一个shell脚本，做个定时任务，但是在shell中用了if语句却不生效，但是手动执行这个shell就生效。

shell:

export PYTHON_HOME="/home/imyin/anaconda2/bin/python"

my_path="$(cd "$(dirname "$0")"; pwd)"
cd ${my_path}

TODAY_DATE=`date +%Y-%m-%d`
trade_date=`cat ${my_path}/trade_date.csv`

if [[ ${trade_date} == *${TODAY_DATE}* ]]; then
echo "Yes..."
else
echo "No..."
fi
