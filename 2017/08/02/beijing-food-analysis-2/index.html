考虑到昨天的分析并不是特别满意，今天和同事商量之后，决定从另一个角度再做分析。

聚类¶

昨天在做特征工程时，对于评论为0的，价格没有的，以及评分没有的商家，直接用0填充。觉得这么做并不好，因为商家的展示页上没有评论并不能够说明其产品质量不好，也许是刚开业或者有其它原因。所以针对以上情况，都统统直接砍掉，再做一次聚类。

# 根据肘部法则来选择k值
from sklearn.cluster import KMeans
from scipy.spatial.distance import cdist

K = range(1,10)
meandistances = []
for k in K:
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(X)
    meandistances.append(sum(np.min(cdist(X,kmeans.cluster_centers_, 'euclidean'), axis=1)) / X.shape[0])
    
plt.plot(K,meandistances,'bx-')
plt.xlabel('k')
plt.ylabel('Average distortion')
plt.title('Selecting k with the Elbow Method')
plt.show()
