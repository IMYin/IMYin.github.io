特征工程¶

此次特征工程比较简单，因为整张表中一共就11个字段，我打算选出6个字段在作为机器学习中的特征变量。

接下要将数据进一步处理，将数据中的price和comment列，去掉其中的中文，并转化为数字类型

缺失数据处理

  1. 对于数据中，没有给出价格的商家，标价统一为0；

  2. 对于数据中，没有给出评价的商家，评分统一为0

# 处理评论数
import re

# 匹配出中文，并且删掉
df["comment_ed"] = df["comment"].apply(lambda x: re.sub("[\u4e00-\u9fa5]+", "", x))
df["comment_ed"] = np.float64(df["comment_ed"].apply(lambda x: str(0) if x == '' else x))

df["price_ed"] = df["price"].apply(lambda x: re.sub("[\u4e00-\u9fa5]+", "", x))
df["price_ed"] = df["price_ed"].apply(lambda x: re.sub("￥", "", x))
df["price_ed"] = np.float64(df["price_ed"].apply(lambda x: 0 if x == "-" else x))

cluster¶

在算法进行之前，我们需要注意一下，KMeans算法虽然好用，但是也存在些缺点，比如其对离异值的敏感度不高（可用基于密度的算法BDSCAN算法处理），在进行聚类之前需要确认k值。

这里我们不考虑离异值的影响，也不是很在乎，大不了就吃亏上当一次，再也不去光顾那家店罢了，好了，再说多就不严谨了，我们继续。

为了确定k值，在这里可以通过肘部法则，来枚举k值，看看在k等于几的时候，畸变程度趋于平稳：

In [12]:
# 根据肘部法则来选择k值
from sklearn.cluster import KMeans
from scipy.spatial.distance import cdist

K = range(1,10)
meandistances = []
for k in K:
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(X)
    meandistances.append(sum(np.min(cdist(X,kmeans.cluster_centers_, 'euclidean'), axis=1)) / X.shape[0])
    
plt.plot(K,meandistances,'bx-')
plt.xlabel('k')
plt.ylabel('Average distortion')
plt.title('Selecting k with the Elbow Method')
plt.show()

从上图可以看出，当k由$1$到$2$时，畸变程度非常大，之后就趋于平稳了，所以在这里k取值为$2$，或者说将商家分类两类：招人待见的和不招人待见的
