什么是TensorFlow¶

上篇文章中我介绍了如何安装TensorFlow，之所以要分开写这两篇文章，是为了以后如果换机器的话能够快速的在我的笔记中找到安装方法。哈哈，那么这篇文章我就详细介绍一下什么是TensorFlow，以及来通过一个小例子说明它的神奇之处。

它是一个可以用数据流图在学习模型的机器学习库，主要的目的是用来解决自然语言处理(NLP)以及图像识别任务。用户可以自己用TensorFlow为深度学习创建一个神经网络结构，并且TensorFlow还允许用户与统计机器学习算法一起使用。但是目前它的使用条件比较苛刻，必须是在它允许的依赖环境中开发程序。此外，Tensorflow是第二代机器学习系统，它已经建立完善并且取代了它的前身DistBelief.

什么是数据流图(data flow graph)¶

在TensorFlow中所有的计算都可以用有向图来表示，它们的computor都是各个节点，通过节点的边缘指向另一个节点。

什么是tensors?¶

数据在TensorFlow中被表示为tensors,简单来理解，它就是个三维的矩阵(当然这么说从数学角度上并不严格)。在下面的图中，你可以看到，程序运行是可以对tensor来回捯饬，相比矩阵而言，它更具有灵活性。

它牛逼在什么地方？¶

  1. 表示起来更加灵活。用户可以通过它来创建各种各样的数据流图，把你的算法用数据流图来展现出来。

  2. TensorFlow支持在CPU上运行和GPU(前提是已经安装了CUDA)上运行,并且它支持并行、异步运算。当然，你可以使TensorFlow运行在任何硬件上，服务器上也中，个人笔记本也中，常言道，你高兴就好。

  3. 你可以用TensorFlow做研究或者生产，再或者，结合一下，你可以用TensorFlow将你做研究用的模型移植到你的产品上，当然在移植之后要注意代码的优化。

  4. 自动分化。TensorFlow还可以为你自动计算导数。如果你喜欢像随机梯度下降这种基于梯度的机器学习算法的话，那么TensorFlow将会给你带来惊喜。

  5. Python接口。这个就不用解释了，爱Python的朋友都懂。当然其他接口也陆陆续续将要到来。

它和其他的机器学习库有什么不一样的地方，比如Theano¶

  • 最大的区别就是TensorFlow支出并行计算，而Theano不行

  • TensorFlow用起来更加灵活

  • TensorFlow的代码结构更加人性化、支持更加简单

一个简单的例子¶

下面这个例子是通过Tensorflow来让机器　学习一个一元线性方程。我会在代码中添加一些注释说明。

In [3]:
import tensorflow as tf
import numpy as np

#随机创造一些随机数

#在tensorflow中大部分数据是float32格式，所以在这里指定一下
x_data = np.random.rand(100).astype(np.float32)
y_data = x_data * 2 + 5  #创建一个一元二次方程

#创建tensorflow结构
Weights = tf.Variable(tf.random_uniform([1],0,10))
biases = tf.Variable(tf.zeros([1]))

#定义将要预测的y
y = Weights * x_data + biases
#定义误差
loss = tf.reduce_mean(tf.square(y-y_data))
optimizer = tf.train.GradientDescentOptimizer(0.5) #创建一个优化器，作用是为了每次迭代学习过程中减小误差，提升参数的准确度
train = optimizer.minimize(loss)
#　初始化变量，让结构“活”起来:Returns an Op that initializes all variables.
init = tf.initialize_all_variables()

#创建会话
# A class for running TensorFlow operations.
sess = tf.Session()
# Runs the operations and evaluates the tensors in `fetches`.
sess.run(init)  

for step in range(100):
    sess.run(train)
    if step % 10 == 0:
        print(step,sess.run(Weights),sess.run(biases))
