安装TensorFlow

Tensorflow的介绍就不过多说了，总而言之它在人工智能领域有很出色的表现。

好了，今天的废话就只说那一句，接下来开始赶紧安装它，我已经迫不及待了。

我在电脑上用的是anaconda，所以我可以用anaconda创建一个虚拟环境来安装Tensorflow,这样在开发Tensorflow的时候也不会和其他项目的依赖发生冲突。

创建虚拟环境¶

conda create -n tensorflow python=2.7

激活虚拟环境¶

source activate tensorflow

当然，如果你没有anaconda的话，也可以用python来创建虚拟环境，但是前提是你已经安装了virtualenv依赖包。

sudo pip install virtualenv

virtualenv 环境名 --no-site-packages    #后面的参数代表这创建一个非常干净的Python环境，没有任何依赖包。

source 环境名/bin/active

激活虚拟环境之后，在命令行的最前面会出现虚拟环境的名称，就像这样：

(tensorflow) sunnyin@sunnyin-win:~$

安装¶

然后执行这个命令，让环境变量生效：

# Ubuntu/Linux 64-bit, CPU only, Python 2.7
(tensorflow)$ export TF_BINARY_URL=https://storage.googleapis.com/tensorflow/linux/cpu/tensorflow-0.11.0rc1-cp27-none-linux_x86_64.whl

当然还有其它选项，可以根据自己的需求进行选择。

在这里需要补充的一点是，tensorflow支持GPU加速，如果你的GPU是英伟达的牌子，在安装的时候就可以选择：

# Ubuntu/Linux 64-bit, GPU enabled, Python 2.7

在安装过程中，它可以自动安装其它依赖。安装完成之后可以测试一下：

pip install --ignore-installed --upgrade $TF_BINARY_URL

安装完成之后会看到这几个依赖包也安装完成了：

six, funcsigs, pbr, mock, setuptools, protobuf, numpy, wheel, tensorflow
