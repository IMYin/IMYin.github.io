"""
上述代码中的 sklearn.linear_model.LinearRegression 是一个估计器，它可以通过观测值来预测结果。
在scikit-learn中，所有的估计器都有fit()和predict()方法。
fit()用来分析模型参数，predict()是通过fit()算出的模型参数构成的模型，对解释变量进行预测获得的值。
LinearRegression类的fit()方法是来学习一元线性回归模型：y=α+βx


一元线性回归假设响应变量和解释变量之间存在线性关系，这个模型所构成的空间就是一个超平面(hyperplane).
超平面是一个子空间，它的维度总是比包围它的空间少一维。
在这个简单线性回归的例子中，一共有两个维度(解释变量和响应变量)，所以超平面就是一维的，是一条线。
如图：
"""
plt.plot(X, y, 'k.')
X2 = [[0], [10], [14], [25]]
model = LinearRegression()
model.fit(X, y)
y2 = model.predict(X2)
plt.plot(X, y, 'k.')
plt.plot(X2, y2,'g-')
plt.axis([0,25,0,25])
plt.grid(True)
plt.title('Pizza price plotted against diameter')
plt.xlabel('Diameter')
plt.ylabel('Price')
plt.show()

#从上图可以清楚地看到residuals的情况，如果residuals最小，那么也就说明拟合效果最好。
#这里用来评估模型拟合度的函数叫做residual sum of squares cost function,就是计算各个差值的平方和。
#利用numpy可以轻松实现：
print 'Residual sum of squares: %.2f' % np.sum((model.predict(X)- y) ** 2)

#现在我们已经有了一个成本函数，接下来寻找模型参数的值来使得它最小化。就是求解一元线性回归的最小二乘法

#评估时用到的是R方( r-squared)
#通过这个模型，R方在响应变量中描述方差的比率一定是介于0至1之间的。
#但是在scikit-learn其它的方法中，当计算的不是用Pearson's r 时，遇到特别差的模型是可能会得到负值。
#在这个例子中，R方是这样计算的：

#首先计算真实价格与价格均值的差值平方和(SStot),再计算出真实价格与测试价格的差值平方和(SSres)
#So, R^2 = 1 - SSres / SStot
#通过scikit-learn 将这样实现：

#首先增加一组测试数据：
X_test = [[8], [9], [11], [16], [12]]
y_test = [[11], [8.5], [15], [18], [11]]
model = LinearRegression()
model.fit(X, y)
print 'R-squared: %.4f' % model.score(X_test,y_test)

y_test = [[8], [12], [15], [18]]
#导入相应的类
from sklearn.preprocessing import PolynomialFeatures
#from sklearn.linear_model import LinearRegression
regressor = LinearRegression()

xx = np.linspace(0,26,100)
yy = regressor.predict(xx.reshape(xx.shape[0],1))

plt.title('Pizza price plotted against diameter')

quadratic_featurizer = PolynomialFeatures(degree=2)
X_train_quadratic = quadratic_featurizer.fit_transform(X_train)
X_test_quadratic = quadratic_featurizer.transform(X_test)

regressor_quadratic = LinearRegression()

xx_quadratic = quadratic_featurizer.transform(xx.reshape(xx.shape[0], 1))
plt.plot(xx, regressor_quadratic.predict(xx_quadratic), c='r',linestyle='--')

#计算R方
print 'Simple linear regression r-squared', regressor.score(X_test, y_test)
print 'Quadratic regression r-squared', regressor_quadratic.score(X_test_quadratic, y_test)
